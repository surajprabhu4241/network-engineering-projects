# Basic LAN Network (Day 1)

## Objective
Build a simple LAN network where devices communicate using a switch.

## Topology
- 1 Switch
- 1 PC
- 1 Laptop

## Configuration
PC:
- IP: 192.168.1.10
- Subnet: 255.255.255.0

Laptop:
- IP: 192.168.1.11
- Subnet: 255.255.255.0

## Testing
Ping test successful between devices.

## Key Learning
- Devices in same subnet can communicate through a switch
- Different subnet requires a router

## Troubleshooting
When IP was changed to 192.168.2.11, ping failed due to different network.
